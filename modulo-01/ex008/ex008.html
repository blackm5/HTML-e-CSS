Principais Formatações

Negrito/Destacado

Nesta frase, temos um termo em negrito usando a tag <b>, (não semântica);
Nesta frase, temos um termo em destaque usando a tag <strong>, (semântica);

Itálico/Êfase

Nesta frase, temos um termo em itálico usando a tag <i>, (não semântica).
Nesta frase, temos um termo em ênfase usando a tag <em>, (semântica).

Marcação

Podemos criar também um texto marcado usando a tag <mark>.
E em outro parágrafo, temos outro texto marcado.

Texto Grande e Pequeno

Criamos um texto grande, usando a tag <big>, (não semântico), e um texto pequeno, usando a tag <small>; neste parágrafo.

Texto Deletado

Podemos marca um texto excluido, usando a tag <del>, para lermos, mas não considera-lo.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins>.
Existe também o sublinhado com a tag <u>(não semântica)

Sobrescrito/Subscrito

eiπ+1=0, com a tag <sup> para sobrescrever.
CO2+H2O = H2CO3, com a tag <sub> para sobscrever.
