Tag <a>: âncora para criar hyperlink 

voltar

  • Parâmetro href: cria uma referência hipertexto.
    Ex.: <a href="ex10.html" >

  • Parâmetro target: controla onde o site de destino vai abrir.

    • _blank: abrir o link em uma nova janela.
      Ex.: <a href="ex10.html" target="_blank" >

    • _self: abrir o link na janela atual.
      Ex.: <a href="ex10.html" target="_self" >

    • _top: desfazer todos os frames e abrir o destino no navegador completo.
      Ex.: <a href="ex10.html" target="_top">

    • _parent:
      Ex.:similar ao uso do _top em uma referência à janela mãe.
      Ex.: <a href="ex10.html" target="_parent">

    • nome-do-freme: caso esteja usando frames, indicar o nome da janela a abrir.

  • Parâmetro rel: indicar qual é a natureza do destino.

    • next: próxima parte do documento.
      Ex.: <a href="ex10.html" target="_blank" rel="next" >

    • prev: parte anterior do documento.
      Ex.: <a href="ex10.html" target="_self" rel="prev" >

    • author: site do autor do artigo atual.
      Ex.: <a href="ex10.html" target="_self" rel="author">

    • external: link para outro site que não faz parte do site.
      Ex.: <a href="ex10.html" target="_self" rel="external" >

  • Parâmetro download: valor configurado para o nome do arquivo.
    Ex.: <a href="arquivos/meulivro.pdf" download="meulivro.pdf" type="application/pdf">


  • Parâmetro type: link direto para o arquivo que se deseja efetuar o download.
    Ex.: <a href="arquivos/meulivro.pdf" download="meulivro.pdf" type="application/pdf" >
