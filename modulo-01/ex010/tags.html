Tag <a>: âncora para criar hyperlink.

  • Parâmetro href: cria uma referência hipertexto. Ex.: <a href="ex10.html">
  • Parâmetro target: controla onde o site de destino vai abrir
    • _blank: abrir o link em uma nova janela. Ex.: <a href="ex10.html"> target="_blank"
    • _self: abrir o link na janela atual. Ex.: <a href="ex10.html"> target="_self"
    • _top:
    • _parent:
