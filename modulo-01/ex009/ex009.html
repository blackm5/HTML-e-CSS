Trabalhando com Listas

Listas Ordenadas

Utilizando a tag <ol> é possível criar listas ordenadas.

  1. Sentar-se;
  2. Ligar o notebook;
  3. Abrir todos os programas necessários;
  4. Iniciar os estudos.

Listas Não Ordenadas

Utilizando a tag <ul> é possível criar listas NÃOordenadas ou, também chamdas, listas de marcadores.

  • Feijão;
  • Arroz;
  • Leite;
  • Carne;
  • Nutella.

Lista de Jogos

Mesclando as duas formas de listas: <ol> + <ul>. Em âmbas é utilizado a tag <li> .

  1. PlayStation
    1. God of War;
      • II;
      • III;
      • Ghost of Sparta.
    2. Castlevania;
    3. GTA San Andreas.
  2. Super Nitendo
    • The Legend of Zelda;
      1. A Link to the Past.
    • Super Mario World;
    • Street Fighter;
    • Megaman.
      1. X.

Lista de Definições

Usando a tag <dl> podemos criar uma lista de definições. com a tag <dt> insere o título a ser definido e com a tag <dd> colocamos a definição.

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
RSA
Sistemas de criptografia de chave pública.
